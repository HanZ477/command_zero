HTML Learning Modules

1. Introduction to HTML

HTML (HyperText Markup Language) is used to create the structure of web pages. It uses elements represented by tags.

<!DOCTYPE html>
<html>
  <head>
    <title>My Page</title>
  </head>
  <body>
    <h1>Hello World</h1>
  </body>
</html>

2. Headings and Paragraphs

Use heading tags <h1> to <h6> and <p> for paragraphs.

<h1>Main Heading</h1>
<p>This is a paragraph.</p>

3. Lists in HTML

HTML supports ordered and unordered lists.

<ul>
  <li>Apple</li>
  <li>Banana</li>
</ul>

<ol>
  <li>First</li>
  <li>Second</li>
</ol>

4. Links and Images

Links are made with <a> and images with <img>.

<a href="https://example.com">Visit Example</a>

<img src="image.jpg" alt="Sample Image">

5. Tables

Tables organize data in rows and columns.

<table>
  <tr>
    <th>Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>John</td>
    <td>25</td>
  </tr>
</table>

6. Forms

Forms collect user input.

<form action="/submit">
  <input type="text" name="name" placeholder="Your Name">
  <input type="submit" value="Submit">
</form>

7. Semantic Tags

Semantic tags describe the meaning of content, such as <article>, <section>, and <nav>.

<nav>
  <a href="#">Home</a>
  <a href="#">Contact</a>
</nav>

9. HTML Entities

Entities are used for reserved characters: &lt; = <, &amp; = &

10. Embedding Media

You can add video and audio using the <video> and <audio> tags.

<video controls>
  <source src="video.mp4" type="video/mp4">
</video>

11. HTML5 New Features

HTML5 introduces new input types like date, email, and range.

<input type="email" placeholder="Enter email">
<input type="range" min="0" max="100">

12. Practice Exercise

Try making your own HTML page with headings, a list, a link, an image, and a table.
